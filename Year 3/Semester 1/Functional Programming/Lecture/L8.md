# Circular Structures

### Fibonnaci sequences 
- define the f_n as sum of previous 2 sequences

```haskell

fibonnaciGen n = take n f0 where
	f0 = 0:f1
	f1 = 1:f2
	f2 = zipWith (+) f0 f1
```

Works like a "factory" of fibonnaci numbers
- the program is a network of streams, synchronization is achieved by lazy evaluation
